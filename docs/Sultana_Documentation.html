SULTANA

Brand Book & Spécifications Techniques

1. Essence de la Marque

Sultana est une plateforme d'empuissancement dédiée aux jeunes filles marocaines. Elle allie la sagesse ancestrale à la modernité technologique pour offrir un parcours éducatif magique.

  • Vision : Un monde où chaque fille connaît et exerce ses droits.
  • Mission : Éduquer par le jeu et le récit interactif.
  • Valeurs : Courage, Savoir, Élégance, Innovation.

2. Identité Visuelle : Le Logo "S"

Le logo "S" est le symbole central de la marque. Il représente Sultana, le Savoir et le Succès.

Concept : Une lettre "S" fluide intégrant une étoile/étincelle pour symboliser l'illumination par la connaissance.

Usage : Utiliser sur fond blanc ou clair pour une lisibilité maximale.

4. Typographie

Outfit - Titres

Utilisée pour les titres et les éléments de navigation. Elle apporte une touche de modernité et de caractère.


Inter - Corps de texte

Utilisée pour tous les textes narratifs et les dialogues. Conçue pour une lisibilité exceptionnelle sur smartphone.

5. Principes d'Interface (UI)

Boutons & Actions

Les boutons doivent avoir un border-radius de 12px. Les actions primaires utilisent des gradients allant du Purple au Turquoise.

Composants "Magiques"

Utilisation du Glassmorphism (flou d'arrière-plan) pour toutes les modales et cartes d'information, créant une sensation de profondeur.

6. Spécifications Techniques

Le projet repose sur une architecture robuste et moderne :

  • Framework : React 19 + Vite
  • Styles : Tailwind CSS 4
  • Navigation : TanStack Router
  • Animations : Framer Motion (Transitions de 0.3s standard)
