I. Cahier des Prescriptions Spéciales (CPS)

Ce document définit les obligations techniques et fonctionnelles pour le développement de la plateforme.

1. Objet et Objectifs

Le projet Sultana vise à créer une application web progressive (PWA) éducative pour les jeunes filles marocaines. L'objectif est de vulgariser les droits juridiques à travers une interface ludique.

2. Spécifications Fonctionnelles détaillées

  • Système d'Univers : Navigation non-linéaire entre 4 modules thématiques.
  • Interactivité : Dialogues à choix multiples avec Sultana (personnage guide).
  • Progression : Sauvegarde locale de l'avancement (Local Storage).
  • Audio : Gestion de l'immersion sonore par nappe atmosphérique.

3. Spécifications Techniques

Le développement doit s'appuyer sur une architecture Single Page Application (SPA) performante.

  • Vitesse : First Contentful Paint (FCP) < 1.2s.
  • Responsivité : Support complet des résolutions de 320px à 1200px (Mobile-First).
  • Accessibilité : Respect du standard WCAG 2.1 niveau AA.
