# SULTANA - Le Parcours des Droits
## Dossier de Spécifications Globales (CPS & Brand Book)

---

# I. Cahier des Prescriptions Spéciales (CPS)

## 1. Objet du Marché
Conception et réalisation d'une application web éducative immersive pour les jeunes filles marocaines.

## 2. Spécifications Fonctionnelles
- **Accueil Immersif** : Audio, visuels animés.
- **Modules Thématiques** : 4 univers (Éducation, Santé, Protection, Futur).
- **Moteur de Dialogue** : Storytelling interactif avec feedback pédagogique.
- **Tableau de Bord de Progression** : Suivi visuel de l'avancement.

## 3. Spécifications Techniques
- **Performance** : Temps de chargement < 1.5s.
- **Référencement** : Optimisation SEO complète.
- **Interopérabilité** : Support mobile Android/iOS (Web-based).

---

# II. Brand Book (Charte Graphique)

## 1. Identité Visuelle
- **Logo principal** : Lettre "S" stylisée avec étoile dorée.
- **Typographie** : Outfit (Titres) et Inter (Corps).

## 2. Design System
- **Couleurs** : Purple (#6D28D9), Gold (#F59E0B), Turquoise (#06B6D4).
- **Composants** : Boutons (radius 12px), Cartes (Glassmorphism).

---

# III. Boîte à Outils (Tech Stack)

## 1. Stack Principale
- **React 19** & **TypeScript**
- **Vite** (Build Tool)
- **TanStack Router** (Navigation)
- **Tailwind CSS 4** (Styling)

## 2. Bibliothèques Secondaires
- **Framer Motion** (Animations)
- **Lucide React** (Icônes)
- **Sonner** (Notifications)

## 3. Infrastructure
- **Hébergement** : Render / Vercel
- **Gestion de Version** : Git & GitHub
